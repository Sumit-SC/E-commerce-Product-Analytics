# Python Analysis Scripts

This directory contains Python scripts for data generation and analysis.

## Scripts

- **data_generator.py** - Generate 500k-1M realistic e-commerce events with noise
- **cohort_analysis.py** - Cohort retention analysis and pivot tables
- **ab_test_analysis.py** - Statistical significance testing for A/B tests
- **utils.py** - Helper functions

## Usage

```bash
# Generate data
uv run python src/data_generator.py

# Run analysis
uv run python src/cohort_analysis.py
uv run python src/ab_test_analysis.py
```
